## Task Description
Implementation of **SPI (Serial Peripheral Interface)** protocol in SystemVerilog.

SPI is a synchronous serial communication interface used for short-distance communication, primarily in embedded systems. It is a full-duplex protocol using a master-slave architecture.

### SPI Protocol Overview
- **SCLK** - Serial Clock (driven by master)
- **MOSI** - Master Out Slave In (data from master to slave)
- **MISO** - Master In Slave Out (data from slave to master)
- **CS/SS** - Chip Select / Slave Select (active low)

### Implementation Details
- SystemVerilog RTL design
- Configurable clock polarity (CPOL) and phase (CPHA)
- Master and Slave modules
- Testbench for verification
